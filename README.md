# IPL 2023 Data Analysis using Python

This project performs exploratory data analysis (EDA) on the IPL 2023 dataset using Python. The analysis focuses on understanding team performance, player statistics, and match outcomes.

## Tools & Libraries Used

- Python
- Pandas
- NumPy
- Plotly / Matplotlib
- Jupyter Notebook

## Objectives of the Project

The following analyses were performed:

1. Number of matches won by each team.
2. Comparison of matches won while batting first vs batting second.
3. Top 10 wicket takers in IPL 2023.
4. Top 10 players with most Player of the Match awards.
5. Top scorers in IPL 2023.

## Key Insights

- Some teams performed significantly better while chasing targets.
- Certain players dominated the Player of the Match awards.
- A few batters scored the majority of runs for their teams.

## Dataset

The dataset contains match statistics and player performance data for IPL 2023.

## How to Run the Project

1. Download or clone the repository
2. Open the Jupyter Notebook file
3. Run the cells to reproduce the analysis and visualizations
